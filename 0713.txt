git upstream是什么，表示本地branch和远程branch之间的关联关系，可以从.git/config处查看
git push <远程主机名> <本地分支名>:<远程分支名>

如果只想取回特定分支的更新，可以指定分支名,如下所示 -
git fetch origin develop/Main

删除远程分支
git push origin --delete 分支名
